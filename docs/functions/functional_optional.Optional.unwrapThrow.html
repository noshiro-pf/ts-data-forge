    Function unwrapThrow

    • unwrapThrow<O extends Optional.Base>(optional: O): Unwrap<O>

      Unwraps an Optional, returning the contained value. Throws an error if the Optional is Optional.None.

      This is a safer alternative to direct value access when you know the Optional should contain a value. Use this method when an empty Optional represents a programming error or unexpected condition.

      Type Parameters

      • O extends Optional.Base

        The Optional.Base type to unwrap.

      Parameters

      • optional: O

        The Optional to unwrap.

      Returns Unwrap<O>

      The contained value if Optional.Some.

      Example

      const present = Optional.some('available');

      assert(Optional.unwrapThrow(present) === 'available');
      assert.throws(
        () => Optional.unwrapThrow(Optional.none),
        /has failed because it is `None`/u,
      );

      Throws

      Error with message "unwrapThrow() has failed because it is None" if the Optional is Optional.None.
