    Type Declaration

      • <E1 extends Primitive, E2 extends Primitive = E1>(

        ): readonly (E1 & E2)[]

      • Returns the intersection of two arrays of primitive types. The intersection contains elements that are present in both arrays. Order is based on array1.

        Type Parameters

        • E1 extends Primitive

          The type of elements in the first array (must be a primitive type).

        • E2 extends Primitive = E1

          The type of elements in the second array (must be a primitive type).

        Parameters

        • array1: readonly E1[]

          The first array.

        • array2: readonly E2[]

          The second array.

        Returns readonly (E1 & E2)[]

        A new array containing elements that are in both array1 and array2.

        Example

        const baseline = [1, 2, 3, 4] as const;
        const removed = [2, 4] as const;

        const remaining = Arr.setDifference(baseline, removed);

        assert.deepStrictEqual(remaining, [1, 3]);

    Example

    const refs = ['Ada', 'Alan', 'Grace'] as const;
    const attendees = ['Grace', 'Alan', 'Barbara'] as const;

    const both = Arr.setIntersection(refs, attendees);

    assert.deepStrictEqual(both, ['Alan', 'Grace']);
