      Casts a string to BoundedLengthString<MinLength, MaxLength> after checking that its length is within the inclusive range [minLength, maxLength].

      Use this instead of writing s as unknown as BoundedLengthString<M, N>: the length is verified at runtime, and the original string type (e.g. literal types) is preserved via intersection.

      Type Parameters

      • S extends string

        The input string type (literal types are preserved).

      • MinLength extends SupportedLength

        The minimum number of characters (inclusive).

      • MaxLength extends SupportedLength

        The maximum number of characters (inclusive).

      Example

      const userId = Str.asBoundedLengthString('user-12345678', 8, 16);

      const relaxed: BoundedLengthString<1, 255> = userId; // OK ([8, 16] ⊆ [1, 255])

      assert.isTrue(relaxed.length >= 8 && relaxed.length <= 16);

      assert.throws(() => Str.asBoundedLengthString('user', 8, 16)); // length 4 < 8

      Throws

      If s.length < minLength or s.length > maxLength.
